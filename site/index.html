ACT 主题包标准

好看之前，先让一套主题值得信任。

标准 .act-theme 是兼容 ZIP 的声明式主题包，只包含 manifest 与已声明素材。包内不允许脚本、远程 CSS 或隐藏运行逻辑。

Codex Native

只做一个目标，而且使用真实的 Codex 主题契约。

Codex 原生支持强调色、背景色、前景色、对比度、字体、代码主题和语义色。原生契约不接受背景图片，因此 Gallery 插画会明确标为封面。
