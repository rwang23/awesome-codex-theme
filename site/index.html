THE CONTRACT

A theme should be inspectable before it is beautiful.

The canonical .act-theme file is a ZIP-compatible package with one manifest and declared assets. JavaScript, shell scripts, remote CSS, and hidden runtime behavior are not allowed inside it.

View JSON Schema ↗
  1. 01

    Declare

    Identity, localized copy, compatibility, motion, provenance, and every asset path.

  2. 02

    Verify

    SHA-256, file size, dimensions, package allowlist, rights fields, and WCAG contrast.

  3. 03

    Export

    Generate engine-specific adapters outside the trusted canonical package.
