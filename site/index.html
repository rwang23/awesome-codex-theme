ACT 主题包标准

好看之前，先让一套主题值得信任。

标准 .act-theme 是兼容 ZIP 的声明式主题包，只包含 manifest 与已声明素材。包内不允许脚本、远程 CSS 或隐藏运行逻辑。

社区共创 / COMMUNITY

社区已经开放，Registry 门禁仍然保留。

独立社区 Beta 支持账号、主题包上传、审核与投票。文件先隔离并校验；票数只表达兴趣，不会自动进入官方 Registry。

进入主题社区 ↗ 查看官方收录路线 ↗
  1. 01 账号与上传

    作者创建账号并上传声明式 .act-theme；主题包不能携带可执行代码。

  2. 02 隔离与校验

    服务端先检查 ZIP 路径、manifest、图片、体积和哈希，失败内容不会公开。

  3. 03 审核与收录

    社区审核通过后仍需走公开仓库的权利、CI 和实机验证，才可进入 Registry。

  4. 04 投票与精选

    每个账号对每套主题一票；高票不会自动成为 Verified 或编辑精选。
